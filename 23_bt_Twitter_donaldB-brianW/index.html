Create a grid by adding a container div.

You can add rows by making row divs as children to the container, and col divs as children to the row divs!

You can make grids have different-sized colums by adding adding the class col-N, with N being the proportion of the space taken.

<div class="container">
        <div class="row">
            <div class="col col-4">
            <div class="col col-8">
            ...

makes a grid with one row and two columns, one column 4/12 of the row and one column 8/12 of the row.

You can make list groups with <ul class="list-group">.

Adding a <li class="list-group-item"> adds a new item to the list group.

Adding the list-group-item-action class to a list-group item makes it hoverable.

Adding the active class makes it active and non-actionable.

Adding the disabled class makes it grayed out and non-actionable too.

By adding an <input type="checkbox" class="form-check-input"> for the checkbox and a <label class="form-check-label"> [LABEL] <label> inside an empty list-item-group, you can make a checklist list group.
